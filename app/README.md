## Stack

- Svelte (**not** sveltekit!)
- Vite
- Tailwind + PostCss
- Shadcn
- Monaco Editor
- Json schema
- Zod (converted to json schema)
- Octokit

<!-- Svelte + vite single page application (SPA) -->

## Commands

### Development

```sh
npm run dev
```

### Check

```sh
npm run check
```

### Build

```sh
npm run build
```

#### Preview build

```sh
npm run preview
```
